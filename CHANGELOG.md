# CHANGELOG

All notable changes to this project will be documented in this file.

## [Unreleased]

### Changed

- Require Nim v1.4.0.

- Add `vls.indent_size`, `vls.tabs_to_spaces` and

- Strip trailing '/' characters from include paths.
